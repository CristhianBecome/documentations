# API de Consulta de Migración Colombia (Compliance)

**Bienvenido a la documentación oficial de la API de Consulta de Migración Colombia.** Esta API permite realizar consultas de migración para Colombia con comparación 1 a 1 de campos contra los registros oficiales. Está diseñada para validar datos de documentos de identidad de migrantes (CE, PPT, PEP) y verificar que la información proporcionada coincida con la registrada.

## Características principales

- ✅ Consulta de migración Colombia con comparación campo a campo
- ✅ Soporte para documentos CE, PPT y PEP
- ✅ Respuestas en tiempo real con resultado booleano por campo
- ✅ Autenticación segura mediante tokens JWT

## URLs base

- **Producción:** `https://api.svi.becomedigital.net/api/v1`
