# API ANI — Registraduría Nacional

**Documentación de la API de verificación ANI contra los registros de la Registraduría Nacional del Estado Civil de Colombia.** Permite integrar la consulta de identidad en aplicaciones con validación frente a los datos oficiales.

Al enviar los datos personales del titular, la API retorna el resultado de la consulta según los registros oficiales colombianos.

## Características principales

- Consulta frente a registros oficiales de la Registraduría
- Respuestas en tiempo real
- Autenticación mediante tokens JWT
- Cumplimiento de requisitos de validación de identidad en Colombia
- Integración sencilla

## URLs base
